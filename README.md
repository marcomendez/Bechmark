# Bechmark

# Command line
**Create a new project**
    dotnet new Project01

**Install Bechmark**
    dotnet add package BenchmarkDotNet --version 0.11.5

**Build project**
    dotnet build -c release

**Run project**
    dotnet run

**Execute Benchmarco**
    dotnet e:\Mio\Bechmark\bin\release\netcoreapp2.1\Bechmark.dll
